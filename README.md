# GREENLAND RUST - V4 image cinématique

Image animée avec effet Ken Burns, brouillard, braises, poussière, lueur de feu, logo progressif, boutons animés, menu animé et ambiance sonore synthétique activable.

Note : je n'ai pas inclus la musique officielle de Rust pour éviter les soucis de droits. Le bouton Ambiance génère un son sombre directement dans le navigateur.

À envoyer sur GitHub : index.html, css/, js/, assets/, README.md. Ne mets pas le ZIP directement.
