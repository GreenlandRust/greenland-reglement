GREENLAND RUST

Règlement officiel du serveur. En rejoignant Greenland, chaque joueur accepte l'intégralité des règles ci-dessous. L'ignorance du règlement ne constitue pas une excuse.

Équipes Alliances Bases PvP & Raids Grief Fair-play Comportement Signalements Sanctions
Important : le staff se réserve le droit d'intervenir sur toute situation non prévue par le règlement lorsqu'un comportement nuit au bon fonctionnement du serveur ou à l'expérience des joueurs.

I. Taille des équipes

La limite est fixée à 3 joueurs maximum par équipe.

Une équipe comprend toute personne autorisée sur :

  • le Tool Cupboard (TC) ;
  • les cadenas ;
  • les tourelles automatiques ;
  • les lits ou sacs de couchage de la base.
Interdit :
  • farmer, construire, défendre ou raid à plus de 3 ;
  • faire intervenir un ancien membre de l'équipe ;
  • contourner cette limite via une base secondaire, un prêt de compte ou une organisation extérieure.

II. Alliances et teaming

Les alliances permanentes sont interdites.

  • Pas de pacte de non-agression permanent.
  • Pas de défense commune organisée.
  • Pas de raid organisé entre plusieurs équipes.
  • Pas de partage régulier de ressources, d'informations ou d'équipements.

Les échanges ponctuels restent autorisés via Shop Front, Vending Machine ou échange rapide. Le staff reste seul juge entre un échange normal et une alliance déguisée.

III. Bases

Chaque équipe doit posséder sa propre base et sa propre organisation.

  • Il est interdit d'héberger une autre équipe.
  • Il est interdit de partager une base, un TC ou une compound.
  • Il est interdit de construire une base dans le but de contourner les limites du serveur.

IV. PvP et Raids

Le PvP est libre. Le kill on sight est autorisé.

Les raids sont autorisés à toute heure.

Pendant un raid :
  • aucune aide extérieure ne peut être demandée ;
  • aucune autre équipe ne peut intervenir volontairement ;
  • les contre-raids naturels sont autorisés uniquement si l'équipe est totalement indépendante.

V. Grief

Le grief est strictement interdit sur Greenland Rust.

Est considéré comme du grief tout comportement visant principalement à empêcher un joueur ou une équipe de continuer à jouer normalement après une action PvP ou un raid.

Actions interdites

  • Condamner volontairement une base.
  • Remplacer un TC dans le seul but de bloquer définitivement les propriétaires.
  • Poser des murs, portes, fondations ou constructions destinés à empêcher durablement la récupération d'une base.
  • Détruire ou modifier une base après un raid sans nécessité de gameplay, uniquement pour nuire.
  • Utiliser une mécanique, un bug ou un glitch pour rendre une base inutilisable.
Après un raid :

Les assaillants peuvent piller la base et sécuriser temporairement leur extraction. En revanche, toute action visant à condamner, bloquer ou rendre définitivement la base inutilisable est considérée comme du grief et sera sanctionnée.

VI. Fair-play

Sont strictement interdits :

  • cheats ;
  • scripts ;
  • macros ;
  • logiciels tiers ;
  • ESP, aimbot, recoil script ;
  • exploitation volontaire de bugs ou glitchs.

Tout bug découvert doit être signalé au staff.

VII. Comportement

Le respect entre joueurs est obligatoire. Aucune tolérance ne sera appliquée concernant :

  • racisme ;
  • homophobie ;
  • discriminations ;
  • harcèlement ;
  • menaces réelles ;
  • usurpation d'identité d'un membre du staff.

Le spam, le flood et la publicité pour d'autres serveurs sont interdits.

VIII. Signalements

Toute plainte doit être accompagnée de preuves exploitables :

  • vidéo ;
  • capture d'écran ;
  • combatlog ;
  • éléments de contexte précis.

Les accusations sans preuve ne seront pas traitées. Les faux signalements volontaires pourront être sanctionnés.

IX. Staff

Les décisions du staff sont finales. Toute tentative de mentir, cacher des informations, falsifier des preuves ou contourner une sanction pourra entraîner une aggravation de la sanction.

X. Sanctions

XI. Bon sens

Toute utilisation d'une faille du règlement ou d'une mécanique dans le seul but d'obtenir un avantage déloyal pourra être sanctionnée à l'appréciation du staff.
